General Ekphrastic Ontology (GEkO)
==================================================
Documentation for the GEkO ontology, version 1 and 2.
<hr>
GEkO is a conceptual model and ontology for describing ekphrasis, i.e. the verbal description of a visual artwork. 
The ontology in its version 2 is integrated within the INTERIM ontology, developed to describe intermedial relations of which ekphrasis constitutes a peculiar example. 
The documentation for INTERIM is availabe at the follwing link: https://github.com/MariaFrancesca6/interim/tree/main .
