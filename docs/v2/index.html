GEkO Ontology v2: Description back to ToC

GEkO Ontology describes the phenomenon of ekphrasis along with technical implementation of image-text relations using annotations.

has Targetop back to ToC or Object Property ToC

IRI: http://www.w3.org/ns/oa#hasTarget

In GEkO it is the property that connects the textual passage to the informational layer. In GEkO the target is double, the first connecting the text to the digital fascimile in which it is contained, the second connecting the text to the digital reproduction of the visual object described. Typically, the two targets address canvas of a IIIF manifest.
